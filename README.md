# Multimodal Fake News Detection System

## Overview
This project detects fake news using a hybrid approach combining:
- Statistical Text Features (TF-IDF)
- Statistical Image Features (Brightness, Resolution, Contrast)

## Model Used
- Logistic Regression

## Technologies
- Python
- Scikit-learn
- OpenCV
- Pandas
- NumPy

## Methodology
1. Text preprocessing
2. TF-IDF feature extraction
3. Image statistical feature extraction
4. Feature fusion
5. Classification

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Run:
   python app.py
